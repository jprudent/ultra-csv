# ultra-csv

A smart reader/writer for CSV files. Right now only reading is implemented.

**HIGHLY EXPERIMENTAL**
**USE AT YOUR OWN RISK**

## Usage

```clojure
(def csv-seq (ultra-csv/read-csv "/my/path/to.csv"))
```
